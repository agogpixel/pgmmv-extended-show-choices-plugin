    • Defined in src/action-commands/show-choices/exec-show-choices-action-command.function.ts:40

    Begin execution of the 'Show Choices' action command 'business' logic.

    Parameters

    • internalApi: PluginProtectedApi

      The plugin's internal API.

    • parameter: Record<number, JsonValue>

      'Show Choices' action command data that is set in & provided by the PGMMV editor or runtime & subsequently normalized.

    • objectId: number

      The object ID of the object instance through which the 'Show Choices' action command is executing.

    • instanceId: number

      The instance ID of the object instance through which the 'Show Choices' action command is executing.

    Returns CommandBehavior

    Action command behavior signal. Usually, Block is returned when we are waiting for a choice to be made (hence this method will be called again on the next frame); Next is returned once a choice is made, or an error occured.

## Requirements

-   [Node.js](https://nodejs.org/en/download/)
    -   Easiest method is the LTS Windows Installer.

## Quick Start

1. Copy this project.
2. `npm install`
3. `npm run build-dev`
4. Copy output file in `dist` to desired location.
5. Import into PGMMV editor.
